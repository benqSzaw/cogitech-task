# COGITECH Recruitment Task

## Vue + Typescript + Vite

### Steps to Start the Project in Development

1. Clone the Repository
  ```bash
  git clone https://github.com/benqSzaw/cogitech-task.git
  cd cogitech-task
  ```
2. Install Dependencies
  ```bash
  npm install
  ```
3. Start project
  ```bash
  npm run dev
  ```
